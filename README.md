# MiniVue

## 介绍

B站up主的视频教学，github地址和B站视频链接如下，侵删。
模板编译、创建组件实例、运行渲染函数、挂载虚拟 dom、接合响应式系统、patch 更新渲染、scheduler 任务调度。
